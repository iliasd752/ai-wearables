AI Wearables 🕶️⌚️

A lightweight React + TypeScript playground for cataloguing the fast‑growing family of AI‑powered wearables — smart glasses, rings, pins, earbuds, and more.

✨ Key features

Interactive catalogue with real‑world examples (Ray‑Ban Meta Glasses, Oura Ring Gen 3, Humane AI Pin).

CRUD UI – add, edit, and delete devices from your personal list.

At‑a‑glance specs: AI capability, form factor, battery life, weight, price, release date, and connectivity.

Modern tooling: Vite 7 + React 19 + TypeScript 5 + Tailwind 4, shadcn/ui components, lucide‑react icons.

Backend‑free – data lives in local component state for ultra‑fast prototyping; swap in a real store or API later.

🏃‍♂️ Getting started

1 · Clone & install

git clone https://github.com/iliasd752/ai-wearables.git
cd ai-wearables
git checkout feature/add-wearable # or main when merged
npm install # or yarn / pnpm

2 · Run the dev server

npm run dev

Open http://localhost:5173 in your browser – hot‑module reload is enabled by default.

3 · Build & preview

npm run build # production bundle in /dist
npm run preview # serve the build locally

🧠 How it works

State management

Wearables live inside a small React state array exposed by useWearablesStore (src/hooks/useWearablesStore.ts).

Persistence – data resets on refresh; plug in localStorage, Zustand, Redux or a remote API if needed.

Extensibility – add new fields by editing Wearable in src/lib/types.ts and updating the form & card components.

Forms & validation

react-hook-form manages form state and validation, while Tailwind + shadcn/ui provide accessible UI elements.

Styling & UI

Tailwind CSS delivers utility‑first styling; lucide‑react supplies crisp SVG icons.

Type safety

Everything is written in TypeScript 5.8 with strict settings.
